Mes compétences

React.js est une bibliothèque JavaScript puissante utilisée pour construire des interfaces utilisateur interactives et dynamiques en se basant sur des composants réutilisables.

Framework Backend - Framework servant d'extension à Node.js permettant la création d'un serveur backend pour les applications web.

Vue.js est un framework JavaScript progressif qui facilite le développement d'interfaces utilisateur intuitives et flexibles grâce à une architecture orientée composants.

Cypress et Playwright sont des outils de test automatisé de bout en bout qui aident à garantir la fiabilité et la performance des applications web en simulant des interactions utilisateur réelles.

Jest est un framework de test JavaScript maintenu par Facebook qui est idéal pour tester le code React, offrant des fonctionnalités comme les tests unitaires, les tests d'intégration et les instantanés.

Mes projets

Epeat est mon projet de fin d'études à Epitech. Le but de ce projet était de créer un site web répertoriant des restaurants, avec des fonctionnalités permettant de voir la distance entre l'utilisateur et les restaurants, d'ajouter des plats au panier et de passer des commandes. J'ai également intégré l'API Google Maps pour afficher l'emplacement des restaurants sur une carte. Pour ce projet, nous avons utilisé React.js pour le front-end et Express.js pour le back-end, en s'appuyant sur Material-UI comme bibliothèque de composants. Nous avons réalisé les tests front-end avec Playwright et les tests back-end avec Jest.

Rotten Tomatoes était un projet de ma formation à Epitech. L'objectif était d'exploiter une API existante qui fournit des listes de films avec des informations détaillées telles que les affiches, les noms et les descriptions. Nous devions afficher ces informations et permettre aux utilisateurs de noter et de commenter les films. Pour ce projet, nous avons utilisé Next.js en combinaison avec Express.js.
